# 📂 Web Calculator 

This repository contains a **minor web calculator project** built using HTML, CSS, and JavaScript. It performs basic arithmetic operations and demonstrates fundamental frontend development concepts.

---

## 📚 About

The purpose of this project is to practice core web technologies by building a functional calculator with a simple and interactive user interface.

---

## 🚀 Features

* Basic arithmetic operations

  * Addition
  * Subtraction
  * Multiplication
  * Division
* Real-time input & result display
* Interactive button-based UI
* Responsive layout design

---

## 🛠️ Tech Used

* HTML5
* CSS3
* JavaScript

---

## 🎯 Learning Outcome

Through this project, I learned:

* DOM manipulation using JavaScript
* Handling user input
* Implementing calculator logic
* Designing UI with HTML & CSS
